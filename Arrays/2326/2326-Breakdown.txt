Final attempt:
- After doing 54, realised the problem would be easier with for loops
- Used a while loop checking the validity of top, bottom, left and right 
- Then within the while loop:
  - For loop writing values to the right, then increment top boundary
  - For loop writing values down, then decrement the right boundary
  - Check the top boundary is valid (aka there are still rows left), then write values left and decrement bottom boundary
  - Check the right boundary is still valid (columns still left), then write the values up and increment left boundary
- In each for loop, check if head is null, if it is insert -1, else insert head.val and set head to the next item
