BEST SOLUTION:
- Go through each item in the array, store the value that is contained at the index
- If the number stored at the index of the value -1 is less than 0 then the value has appeared before
- Else negate the number stored at value - 1 
- E.g. [1,3,4,2,2] -> [-1,-3,-4,..,..] at this point nums[3 - 1] = 3, and thats been set to -3, now when nums[4] = 2 again, nums[2 - 1] is already negative (-3) so 2 is repeated
